# Uphold payments by [Payzing](https://payzing.io) - WooCommerce Gateway

Start accepting instant global payments with no chargebacks for just 0.25%. The most accessible payment method for your customers around the world.

Payzing makes it simple for online consumers and merchants to make and receive payments using Uphold, the world's leading cloud-based financial service that makes holding, sending and receiving money as easy, instant and free as email.  Uphold publishes a real-time proof of solvency and liquidity so you know your money is safe and accessible at all times.

Note. requires [https://uphold.com/](https://uphold.com/) account, sign up for free today.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Payzing.io - WooCommerce Gateway screen to configure the plugin
4. Once in the settings you must: Tick to enable the gateway as an option, enter your Uphold username, enter your Gateway API key which can be found from the help guides section of the Payzing admin.
